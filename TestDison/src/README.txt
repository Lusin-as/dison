README for TestDisoon
Author/Contact: trang.cao@upf.edu

Description:

TestDison is a demo of DISON - a generic management framework.AppC.nc.

Tools:

The Java application displays readings it receives from motes running the
TestDison demo via a serial forwarder. To run it, change to the
./java subdirectory and type:

  make
  export MOTECOM=serial@portnumber:mote_name
  ./run
